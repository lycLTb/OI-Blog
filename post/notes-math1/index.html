数论分块

是非常常用的东西 所以有必要讲一讲

数论分块适用于这样的形式：

∑i=1n⌊ni⌋\sum\limits_{i=1}^{n} \left\lfloor \frac{n}{i} \right\rfloor i=1∑n​⌊in​⌋

我们发现有许多的 ⌊ni⌋\left\lfloor \frac{n}{i} \right\rfloor⌊in​⌋ 值是相同的，相同的值形成了一个个连续的块，所以可以用数论分块来加速求和过程，复杂度 O(n)\mathcal{O(\sqrt{n})}O(n​) 。

附代码

for(int l=1,r;l<=n;l=r+1){
	r=n/(n/l);
	ans+=(r-l+1)*(n/l);
}

其中 ans+=(r-l+1)*(n/l); 的意义为 有长度为 r-l+1 的块的值为 n\l 。

顺带一提，这个式子可以化为

∑i=1n⌊ni⌋=∑k=1nd(k)\sum\limits_{i=1}^{n} \left\lfloor \frac{n}{i} \right\rfloor = \sum\limits_{k=1}^{n}d(k) i=1∑n​⌊in​⌋=k=1∑n​d(k)

这里 d(k)d(k)d(k) 是前文提到过的表示约数个数的函数

由于 ddd 可以用欧拉筛预处理出来，而这个式子就是 ddd 的前缀和，所以可以在线性时间内预处理出来。

当然，并不是所有可以用数论分块解决的问题都可以这样在线性时间内预处理（比如在求和的时候乘个什么东西 是很常见的形式），这就是数论分块的意义所在。

Dirichlet 卷积

定义

设 f,gf,gf,g 是数论函数，且数论函数 hhh 满足

h(n)=∑d∣nf(d)g(dn)h(n)=\sum\limits_{d|n}f(d)g(\frac{d}{n}) h(n)=d∣n∑​f(d)g(nd​)

则称 hhh 为 f,gf,gf,g 的 Dirichlet 卷积， 记作 h=f∗gh=f*gh=f∗g

对于任意数论函数 fff ，有 f=f∗ϵ=ϵ∗ff = f * \epsilon = \epsilon * ff=f∗ϵ=ϵ∗f ，即单位函数 ϵ\epsilonϵ 是 Dirichlet 卷积的 单位元。
