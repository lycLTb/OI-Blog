插板法

经典小学奥数内容

把 nnn 个相同的球放进 mmm 个不同的盒子里，每个盒子里至少要有一个球，求方案数。

考虑在 n−1n-1n−1 个空隙中插入 m−1m-1m−1 块板子将 nnn 个球分成 mmm 分，显然方案数为 (n−1m−1)\dbinom{n-1}{m-1}(m−1n−1​)。

我们现在考虑每个盒子里不一定要都有球的情况。

我们先给每个盒子钦定一个球，于是我们把球的总数加上 mmm，易得方案数为 (n+m−1m−1)\dbinom{n+m-1}{m-1}(m−1n+m−1​)

Catalan 数

いっそ僕の全部、カトレア

我们定义 Catalan 数

Cn=1n+1(2nn)=(2n)!(n+1)!n!C_n = \frac{1}{n+1} \binom{2n}{n} = \frac{(2n)!}{(n+1)!n!} Cn​=n+11​(n2n​)=(n+1)!n!(2n)!​

一个 Catalan 数的经典组合意义是一个长度为 nnn 的序列的合法出栈序列个数。

我们定义出栈序列是一个 010101 序列，其中 000 表示入栈，111 表示出栈，000 和 111 各有 nnn 个。

显而易见的，一个合法的出栈序列的每一个前缀中 000 的个数总是不小于 111 的个数——也就是说，对于每一个 1≤i≤n1 \le i \le n1≤i≤n，有 ∑j=1i[aj=0]≥∑j=1i[aj=1]\sum\limits_{j=1}^i [a_j=0] \ge \sum\limits_{j=1}^i [a_j=1]j=1∑i​[aj​=0]≥j=1∑i​[aj​=1]（其实也可以把这个当作它的定义）

我们可以证明，每一个不合法的出栈序列与每一个由 n+1n+1n+1 个 000，n−1n-1n−1 个 111 组成的长度为 2n2n2n 的 010101 序列一一对应——通过把不合法出栈序列的第一个不合法的前缀进行 010101 反转得到。

举个例子： 001110001110001110 是一个不合法的出栈序列，001110011100111000 是它的第一个不合法的前缀（即 111 的个数大于 000 的个数），于是我们对其 010101 翻转，即将原序列变为 110001100011000000。

无根树 / Prufer 序列

<课件>

01 无根树转 Prufer 序列

每次选择一个编号最小的叶结点并删掉它，然后在序列中记录下它连接到的那个结点。重复 n−2n-2n−2 次后就只剩下两个结点，算法结束。

  • Prufer 序列的长度为 n−2n-2n−2
  • 每个点在 Prufer 序列中出现的次数为其度数-1

02 Prufer 序列转无根树

每次选择一个度数为 111 的最小的编号节点，与当前枚举到的 Prufer 序列的点之间连上一条边，然后同时将两个点的度数-1。最后在剩下的两个度为 111 的点间连一条边。
注意到，点数为 nnn 的无根树与长度为 n−2n-2n−2 的 Prufer 序列一一对应

</课件>

于是树的计数的一般思路就是对 Prufer 序列计数

注意到要求的实际上就是长度为 n−2n-2n−2，值域为 1∼n1\sim n1∼n 的 Prufer 序列的数量，于是方案数即为 nn−2n^{n-2}nn−2
